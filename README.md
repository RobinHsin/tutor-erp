# Tutor ERP System

A tutoring center management system designed for education businesses.

This project helps tutoring schools manage students, courses, and enrollments in a simple and organized way.

---

## Features

* Student management
* Course management
* Enrollment system
* Basic CRM functionality

---

## Tech Stack

Backend

* PHP
* MySQL

Frontend

* HTML
* CSS
* JavaScript

---

## System Modules

Students

Manage student information and records.

Courses

Create and manage course schedules.

Enrollments

Enroll students into courses.

Dashboard

Overview of system information.

## Purpose

This project demonstrates how to build a simple education ERP system using PHP and MySQL.
